# Assets Structure

Esta carpeta contiene todos los recursos gráficos y multimedia del proyecto.

## Estructura

- `images/`: Imágenes generales del sitio
  - `hero/`: Imágenes principales de cada sección
  - `backgrounds/`: Fondos y texturas
  - `gallery/`: Imágenes para galerías
  - `events/`: Imágenes específicas de eventos

- `icons/`: Iconos del sistema
  - `ui/`: Iconos de interfaz
  - `social/`: Iconos de redes sociales
  - `brand/`: Iconos de marca

- `logos/`: Logos y marcas
  - `main/`: Logo principal
  - `variants/`: Variantes del logo
  - `partners/`: Logos de partners

- `fonts/`: Fuentes personalizadas
  - `primary/`: Fuente principal
  - `secondary/`: Fuente secundaria

## Convenciones

1. Nombrado de archivos:
   - Usar minúsculas
   - Separar palabras con guiones
   - Incluir tamaño/resolución cuando sea relevante
   - Ejemplo: `hero-main-1920x1080.jpg`

2. Formatos:
   - Imágenes: JPG (fotos), PNG (transparencia), WebP (optimizado)
   - Iconos: SVG (preferido), PNG (fallback)
   - Logos: SVG (preferido), PNG (fallback)
   - Fuentes: WOFF2, WOFF, TTF

3. Optimización:
   - Todas las imágenes deben estar optimizadas
   - Usar herramientas como ImageOptim o TinyPNG
   - Mantener versiones @2x y @3x cuando sea necesario

4. Organización:
   - Agrupar recursos por sección/página
   - Mantener nombres descriptivos
   - Documentar cambios importantes
